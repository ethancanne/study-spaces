Class: StudyGroupFeed

StudyGroupFeed(userSchema)

Provides an interface for working with users in the database.

Constructor

new StudyGroupFeed(userSchema)

Initializes the user to the account passed in from the database.
Parameters:
Name Type Description
userSchema Mongoose.Schema The database record for a given user.
Properties:
Name Type Description
areaCode String The user's area code.
conversations Array.<String> The user's conversations/chats. This is stored as a list of MongoDB document IDs so that the conversations can be accessed directly from the user.
email String The user's email address.
name String The user's name.
passwordHash String The user's hashed password.
profilePicture Buffer The user's profile picture. This must be less than 16MB.
studyGroups Array.<String> The study groups the user is a part of. This is stored as a list of MongoDB document IDs so that the study groups can be accessed directly from the user.
